# Грант Авто Landing

Одностраничный лендинг автосервиса «Грант Авто» в Балашихе, мкр. Железнодорожный: SEO-страница, конверсионные CTA, форма записи, юридические страницы и статическая сборка для GitHub Pages.

## Как открыть локально

Для разработки Next.js:

```bash
npm install
npm run dev
```

Сайт откроется на `http://127.0.0.1:3000`.

Для проверки статической версии GitHub Pages:

```bash
npm run build:pages
npx serve .
```

После `build:pages` в корне проекта появится `index.html`, `_next/`, юридические страницы и `.nojekyll`.

## Форма заявки

В обычном Next.js-режиме форма отправляет данные в `POST /api/leads`. Валидные заявки сохраняются в `data/leads.jsonl`; Telegram-уведомления включаются через переменные:

```bash
TELEGRAM_BOT_TOKEN=
TELEGRAM_CHAT_ID=
```

В статической версии для GitHub Pages backend недоступен, поэтому форма не имитирует отправку. После проверки обязательных полей показывается безопасное сообщение с просьбой позвонить по номеру `8 (800) 505-65-67` или дождаться подключения формы.

## Проверка перед публикацией

```bash
npm test
npm run lint
npm run build
npm run build:pages
```

## Деплой на GitHub Pages

1. Выполнить `npm run build:pages`.
2. Убедиться, что в корне есть `index.html` и `.nojekyll`.
3. Закоммитить исходники и статические файлы сборки.
4. В настройках репозитория GitHub Pages выбрать ветку `main` и папку `/root`.
5. После публикации открыть сайт и проверить форму, телефон, маршрут, футер и юридические ссылки.

## Что заполнить вручную

В юридических блоках оставлены плейсхолдеры `[указать]` для полного наименования исполнителя, ИНН, ОГРН/ОГРНИП, юридического адреса и e-mail. Их нужно заменить только на подтверждённые данные.
